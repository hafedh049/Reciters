## 🎧 About

QuranicAudio Player is a platform designed to provide easy access to beautiful recitations of the Holy Quran. Our mission is to make the recitations of the Quran accessible to everyone around the world through a user-friendly interface to browse different reciters and surahs.

## ✨ Features

- Browse through different categories of recitations
- Search for specific reciters
- Listen to high-quality audio of Quranic recitations
- Dark theme with beautiful particle animations
- Responsive design that works on all devices

## 🚀 Technology

This application is built using:

- Next.js
- React
- Tailwind CSS
- HTML5 Canvas for particle animations

## 📊 Data Source

The audio files and data are sourced from the QuranicAudio.com API. We are grateful for their service in making these beautiful recitations available.

## 🌟 Key Features

### Recitation Categories
- Standard Recitations
- Recitations from Haramain Taraweeh
- Non-Hafs Recitations
- Recitations with Translations

### Audio Player
- Play/pause functionality
- Skip to previous/next surah
- Repeat and shuffle options
- Volume control
- Beautiful circular progress indicator

### Quran Text Display
- Read along with the recitation
- Arabic text with verse numbers
- Highlighted verses that follow the audio playback

### Integration with Quran.com
For a complete Quranic experience, the application integrates with Quran.com for reading translations and tafsir.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/quranic-audio-player.git

# Navigate to the project directory
cd quranic-audio-player

# Install dependencies
npm install

# Start the development server
npm run dev
```
